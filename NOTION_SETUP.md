## 「地域」入力ルール

カードの画像右下と、モーダル内に「地域」バッジとして表示される項目。粒度を揃えることで、利用者が開催場所を一目で把握できる。

**国内大会**: `{都道府県} {市区町村}` の形式
- 例: `東京都 新宿区` / `東京都 文京区` / `大阪府 大阪市中央区`
- 都道府県と市区町村の間は**半角スペース**

**国際大会**: `{国名} / {都市名}` の形式（英語表記）
- 例: `Italy / Cervia` / `Slovenia / Koper` / `Germany / Berlin`
- 国名と都市名の間は **ハイフンではなくスラッシュ（ / ）**
- 開催地が未定の場合は `Europe / TBA` のように大陸名 + TBA

※「地域」は**カードでひと目で見える唯一の場所情報**なので、必ず入力する。未入力の場合は地域バッジが非表示になる。

## Netlify への接続設定（初回のみ）

1. [Notion Developers](https://www.notion.so/my-integrations) で Integration を作成
2. 生成された API Key (`secret_xxx...`) をコピー
3. データベースを開き、右上「...」→「接続」→ 作成した Integration を追加
4. **Netlify Dashboard** > Site settings > Environment variables に追加:

| キー | 値 |
|---|---|
| `NOTION_API_KEY` | `secret_xxxxx...` |
| `NOTION_DATABASE_ID` | `7478a4a53e4b42c18359fe336f9546ee` |

5. Netlify でサイトを再デプロイ

## 大会情報の更新方法

### 新しい大会を追加する
1. データベースに新しい行を追加
2. 各プロパティを入力
3. `公開ステータス` を **`公開`** に設定 → 即時サイトに反映（最大5分）

### 大会を非表示にする
- `公開ステータス` を `非公開` または `準備中` に変更

### フライヤー画像を表示する
- 画像をどこか（Google Drive、Cloudinaryなど）にアップロードして外部URLを取得
- `バナー画像URL` または `サムネイル画像URL` に貼り付け

> **注意**: Notion内部にアップロードした画像はURLが7日で失効します。
> 外部ストレージのURLを使うことを推奨します。
