大会ルール

IDSF Japanが主催する競技会の規定です。参加前に必ずご確認ください。

01

大会の種類

Competition Types

IDSF Japanでは、レベルや目的に合わせた4種類の競技会を開催しています。

初級〜中級

Syllabus Cup

競技を始めたばかりのカップル向け大会。各クラスで定められた基本フィギュアのみを使用して演技します。クラスD〜B1のシラバスレベルのカップルが対象です。

全レベル参加可

Star Cup

入門クラスからMasterクラスまで、全レベルのカップルが参加できる大会です。シニア専用のSenior Cupも開催されます。

シーズン末・3月

Championships

シーズン終了（3月末）に開催される選手権大会。Syllabus ChampionshipsとStar Championshipsがあり、年間ランキングの最終決定と上位クラスへの昇格審査が行われます。

国際大会

International Championships

Italian Open等のIDSF本部直轄の国際選手権大会です。競技キャリアにおける重要なステップとなります。

02

シーズン構成

Season Structure

4月1日に開幕し、翌年の3月31日をもってシーズンが終了します。シーズン中は各地でSyllabus CupとStar Cupが開催され、3月のChampionshipsで年間王者が決定します。

期間 イベント 内容
4月〜2月 Syllabus Cup / Star Cup 年間通じてポイントを獲得。各地で複数回開催
3月末 Syllabus / Star Championships シーズン最終大会。年間ランキング確定・昇格審査
毎年1月1日 カテゴリー更新 年齢カテゴリーが更新される（最年長パートナーの年齢を基準）

昇格ルール

  • 指導者（先生）の申請によりいつでも上位クラスへの昇格が可能
  • Championshipsでの上位入賞（参加人数により異なる）により昇格が義務づけられる場合あり
  • 規定のランキングポイントに達した場合も自動的に昇格となる
  • 降格は原則認められない
05

シラバス種目

Permitted Figures

各クラスで使用できるフィギュア（ステップ）は、クラスに応じて制限があります。Bronze・Silver・Goldの3レベルが基準です。

クラス 使用可能フィギュア 特記事項
Class D / C Bronze のみ Viennese WaltzはBronzeフィギュアのみ（全クラス共通）
Class B3 / B2 Bronze + Silver D/Cクラスでは最大3つのSilverフィギュアでBronzeをつなぐことが可能
Class B1 Bronze + Silver + Gold 全3レベルのフィギュア使用可
Class A 以上 制限なし フィギュアの制限なし。自由な振り付けが可能

シラバスインビジレーター（監視員）が振り付けの規則遵守を確認します。違反した場合は口頭警告、当該種目の失格、または大会全体の失格となる場合があります。

06

ドレスコード

Dress Code

衣装はカテゴリーおよびクラスによって規定が異なります。低年齢カテゴリーほど制限が厳しく、上位クラスでは自由度が増します。

衣装

  • ワンピースと同色のキュロット（一体型インナーパンツ）、またはトップス＋スカート
  • もしくは黒スカート＋白または黒のトップス／ブラウス

上半身

  • 同色なら異素材の組み合わせOK
  • レースやメッシュの透け素材使用可（袖や胸上部は完全透けOK）
  • ヒップ周りの細かいプリーツや背中ボタン（同色のみ）OK
  • 襟・袖は規定の形のみ可

スカート

  • 無地またはプリーツ（1〜3ハーフサークル）
  • 丈：膝上5cm〜足首まで
  • 同色の三角形インサート（レース・透け素材）可
  • 同色のアンダースカートOK
  • 裾にナイロン糸、ヘアクロス、サテン仕上げ、フリル可
  • 使用素材は同色のみ（金属・完全透け・肌色はNG）

シューズ

  • スタンダード用シューズ（色自由）
  • 白い短いソックス、または肌色のソックス・タイツ着用可

07

審査方式

Judging System

全ての審査はInternational Skating System（ISSシステム）を採用しています。

5名以上
審査員は5名以上の奇数で構成
50%以上
各ラウンドで上位50%以上が次ラウンドへ進出
6〜10組
決勝出場組数（同点の場合は最大10組）

公正性の確保

  • 審査員は自身の生徒・2親等以内の親族を審査することはできない
  • ラウンド間で審査員のローテーションを行う場合がある
  • 大会ディレクターは規則に定めのない事態に対して裁量権を持つ
08

エントリー方法

Entry Requirements

参加には公式ポータルへの登録と期日内のエントリー・支払いが必要です。

Step 01

ポータル登録

公式ポータルサイトへの会員登録を行います。カテゴリー・クラスはポータルの登録情報に基づき決定されます。

Step 02

エントリー申請

大会初日の15日前までにエントリーと支払いを完了してください。期日を過ぎた場合は追加料金が発生します。

Step 03

参加確認

エントリー受理後に参加確認のご連絡をします。当日は受付にて参加証を提示してください。

参加資格

  • カップルは男性1名・女性1名で構成されること
  • 参加者全員が公式ポータルに登録済みであること
  • 現在のカテゴリー・クラスで登録されていること

ご不明な点はお問い合わせページよりご連絡ください。

参加を検討中の方へ

入会は無料です。まずはお気軽にお問い合わせください。
